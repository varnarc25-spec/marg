## How to Check Which Account is Logged In

To see which Firebase account the FlutterFire CLI is currently using, run:

```bash
# Check current login status
flutterfire login --list

# Or check Firebase CLI directly
firebase login:list

# Check current project
firebase projects:list
```

## To Switch Firebase Accounts

If you need to use a different Firebase account:

1. **Logout from current account:**
   ```bash
   flutterfire logout
   # or
   firebase logout
   ```

2. **Login with different account:**
   ```bash
   flutterfire login
   # or
   firebase login
   ```

3. **Select the correct project:**
   ```bash
   flutterfire configure
   ```
